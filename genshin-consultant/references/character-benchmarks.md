# Character Benchmarks

These are consulting ranges, not universal truth. Always adjust by character, team, weapon, artifact set, and current guide sources.

## General priority order

1. Mechanics/rotation correctness: stacks, marks, stance windows, Bond of Life, summons, reaction ownership, buff pickup, or other kit-specific actions.
2. ER/rotation stability: burst or skill cycle must work before optimizing crit.
3. character and weapon level.
4. key talent levels.
5. artifact main stats and set effect.
6. useful substats.
7. luxury optimization, constellations, and signature weapon.

## Target tier language

Use these four tiers whenever the user asks “몇까지 맞춰야 해?” or “이 정도면 좋아?”

| Tier | Meaning | Consulting tone |
|---|---|---|
| 최소 실전 | Clears ordinary content if main stats and rotation are correct. | “일단 굴러감” |
| 안정권 | Comfortable default target for most players. | “여기까지 먼저” |
| 준종결 | Strong investment; good stop point for non-favorites. | “여기서 멈춰도 됨” |
| 종결/고점 | Resin-heavy luxury optimization. | “애정/고점용” |

## Crit and conditional-stat interpretation

- CR/CD targets must account for conditional crit from sets, weapons, resonance, passives, and teammates.
- If using Marechaussee Hunter, Blizzard Strayer, Obsidian Codex, or other crit-granting sets, show both sheet crit and effective crit and warn about overcap.
- For characters with non-crit scaling or transformative reaction ownership, do not over-prioritize crit.
- If the weapon gives large crit, lower artifact crit pressure and highlight ER/EM/main-stat needs instead.

## ER target rules

Never give a single ER number without context. ER depends on:

- team particles and batteries
- Favonius/Sacrificial weapons and proc consistency
- rotation length
- whether burst is mandatory every rotation
- enemy particle generation
- constellation or passive energy changes

When exact team context is missing, give a range and label it `team-dependent`. When roster/team is known, give `minimum ER for this team` and explain how to lower it.

## Interpretation rules

- If ER is too low, fix ER before chasing crit damage.
- If mechanics are misunderstood, fix usage before farming artifacts.
- If a transformative reaction trigger is below level 90, level is often the highest-value upgrade.
- If artifact main stats are wrong, replace them before evaluating substat quality.
- If the screenshot lacks team context, avoid exact ER targets and provide a range plus needed team info.
- If a character has multiple roles, give separate target cards per role rather than one mixed target.
